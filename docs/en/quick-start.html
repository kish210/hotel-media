📺 SignageCMS — Quick Start

← Full Documentation

1. Install

cd D:\duc
Expand-Archive signage-cms-final.zip -DestinationPath . -Force
cd signage-cms
docker compose up -d --build

2. Run Migrations

Get-Content "database\migrations\001_complete_schema.sql" -Raw -Encoding UTF8 | docker exec -i signage_mysql mysql -u signage_user -pStrongPassword123! signage_cms
Get-Content "database\migrations\003_module_tables_only.sql" -Raw -Encoding UTF8 | docker exec -i signage_mysql mysql -u signage_user -pStrongPassword123! signage_cms

3. Login

URL: http://localhost
Email: admin@signagecms.com
Password: Admin@123456

⚠ Change password after first login

Key URLs

  • Admin Panel: http://localhost
  • Documentation: http://localhost/docs
  • Player: http://localhost/player/SCR001
  • WebSocket: ws://localhost:8080

TV Activation

  1. Open http://[SERVER-IP]/player/[SCREEN-CODE] on TV browser
  2. From admin panel → Screens → Screen → Activation tab → Get new code
  3. Enter 6-digit code on TV
